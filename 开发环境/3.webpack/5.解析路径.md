解析路径
===

webpack打包文件时，能解析文件内的三种路径：
1.绝对路径
```
import '/file';
```
2.相对路径
```
import './file';
```
3.模块路径
```
import 'module'

import { createApp } from 'vue'
```
webpack在寻找时，默认在 resolve.modules 中(node环境就是node_nodules文件夹)的所有目录中检索模块。

### resolve
帮助寻找模块的配置。

**resolve.alias**
给路径起名，后续使用该路径时，就可以使用别名替代。
```
resolve: {
    alias: {
        // 路径上上级文件夹可以简写为@符号
        @: '../..'
    },
},
```

**resolve.extensions**
配置模块默认后缀，解析时按顺序尝试，自动添加后缀名。
```
resolve: {
    extensions: ['.js', '.json', '.vue'],
},
```
