文件夹
===
###os模块
os模块用于与操作系统交互。
```
import os
# 创建目录
os.mkdir('./testDir')
# 删除目录
#os.rmdir('./testDir')
```
**路径**
不同操作系统路径分隔符不同，所以兼容性的路径应该由`os.path.join(path1,path2)`函数拼接而成。
拆分路径时，也不要直接去拆字符串，而要通过os.path.split()函数，这样可以把一个路径拆分为路径和文件两部分。
```
os.path.split('/Users/testdir/file.txt')
# ('/Users/testdir', 'file.txt')
```
