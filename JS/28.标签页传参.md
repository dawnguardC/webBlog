标签页传参
====

### BroadCast Channel
BroadcastChannel 接口用于同源的标签页之间或者frame之间相互通信。消息通过 message 事件进行广播。

BroadcastChannel 接口创建一个广播频道，其他同源标签页会监听该频道：
```
let channel1 = new BroadcastChannel('myChannel'); // 创建广播
channel1.postMessage('Hello');
```
其他页面接收消息：
```
let channel2 = new BroadcastChannel('myChannel');
channel2.onmessage = function(event) {
  console.log(event.data); //Hello
};
```
close()方法，会不再接收信息并垃圾回收channel。
```
channel2.close();
```

### localStorage / sessionStorage
同源的标签页之间也可以使用 storage 通信，当存储区域（localStorage 或 sessionStorage）被修改时，window对象将触发 storage 事件。
