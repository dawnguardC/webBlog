理解gulp
===================
Gulp是一个自动任务运行器。它充分借鉴了Unix操作系统的管道（pipe）思想，可以通过设置配置文件，来自动化的**合并**，**压缩**，**监听** js/css/sass/html/image文件。

###安装gulp

全局安装 gulp：

    $ npm install gulp -g

作为项目的开发依赖（devDependencies）安装：

    $ npm install gulp -D
如果因为网络原因安装失败，也可以使用淘宝镜像进行安装：

    $ npm install -g cnpm --registry=https://registry.npm.taobao.org
再安装模块的时候, 就能直接从 淘宝 NPM 安装了

    $ cnpm install [name]

###建立配置文件

在项目根目录下创建一个名为 gulpfile.js 的文件，并通过`require()`引入gulp模块：

    var gulp = require('gulp');
这样，就可以使用gulp模块了。
