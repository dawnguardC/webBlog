自定义指令
===================
除了核心功能默认内置的指令，Vue 也允许注册自定义指令。

**全局指令**

    Vue.directive('name', {})

**局部指令**

    directives: {
        name: {}
    }
其中，`{}`可以是函数或者钩子函数，如果使用函数，则：

    Vue.directive('name', function(el,binding){})
el表示指令所在的标签属性，binding表示指令相关数据对象（可以console.log(binding)具体查看）。
如果使用钩子函数，则：

    Vue.directive('name', {
        钩子函数名:function(el,binding){}
    })
